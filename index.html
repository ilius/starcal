StarCalendar

A full-featured international calendar written in Python

Dependencies for StarCalendar 3.1

Python 3.x (>= 3.5)
python3-gi (gobject-introspection binding)
python3-gi-cairo
python3-httplib2
python3-dateutil
python3-psutil
python3-requests
python3-igraph (optional)

Dependencies for StarCalendar 2.4

Python 2.7
PyGTK
python-gobject (old binding)
python-httplib2
python-igraph (optional)
python-dateutil (optional)
