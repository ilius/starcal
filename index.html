StarCalendar

A full-featured international calendar written in Python

Dependencies for StarCalendar 3.2.x

Python 3.x (>= 3.8)
python3-gi (gobject-introspection binding)
python3-gi-cairo
python3-httplib2
python3-dateutil
python3-psutil
python3-requests
python3-igraph (optional)
gtksource-4 / gtksourceview4 (optional)
ntpdate (optional)
python3-pygit2 (optional)
